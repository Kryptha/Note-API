# Welcome!

Note API is a simple API with base CRUD functions. The objective of this new project is to practice connection to a React APP and will add new features soon with new gems. 

# Technologies
* Ruby 3.0.4
* Docker 20.10
* Ruby on Rails 7

# Features
* Create a note 
* Update a note 
* Read a note
* Delete a note

# Features coming soon
* Add tags 
* Add Image 
* User Authentication
